# Builder Memory Log

> **Purpose:** Every implementation action and its outcome is logged here. Before writing any code, the Builder MUST read this file and ensure past mistakes are not repeated.

## Format

Each entry follows this structure:

```
### [YYYY-MM-DD] <short title>
- **Action:** What was implemented / changed
- **Consequence:** What happened as a result (positive or negative)
- **Lesson:** What to do (or avoid) next time
- **Severity:** low | medium | high | critical
```
